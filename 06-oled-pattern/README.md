# 06-oled-pattern

Displays test patterns on the SSD1306 OLED: solid fill, vertical/horizontal stripes, checkerboard, gradient, border, and a font sample.

![Demo](https://github.com/user-attachments/assets/16695622-e751-42d2-aedc-238c484a22ce)

## What you'll learn

- Pixel-level framebuffer manipulation
- Drawing geometric patterns with setPixel
- Full ASCII font rendering

## Wiring

![Wiring diagram](https://github.com/user-attachments/assets/322da340-24b0-4728-b386-96d940168c1b)

| Pico | SSD1306 |
| ---- | ------- |
| GP16 | SDA     |
| GP17 | SCL     |
| 3V3  | VCC     |
| GND  | GND     |
